# README — BloomChain 🌸🔗

## Descripción del Proyecto

BloomChain es un sistema hidropónico inteligente que combina agricultura sostenible con tecnología blockchain. Nuestro proyecto utiliza macetas hidropónicas horizontales equipadas con sensores IoT capaces de monitorear temperatura, humedad, pH y nutrientes en tiempo real.

Toda la información del cultivo se almacena de forma segura mediante blockchain, permitiendo trazabilidad completa desde la siembra hasta la cosecha.

BloomChain busca transformar la agricultura urbana mediante una solución moderna, ecológica y tecnológica que reduzca el consumo de agua y aproveche espacios pequeños.

# Problema 

La agricultura tradicional presenta varios problemas:

* Alto consumo de agua.
* Necesidad de grandes extensiones de tierra.
* Contaminación ambiental.
* Dificultad para cultivar en ciudades.
* Falta de trazabilidad y monitoreo inteligente.

Además, muchas personas en zonas urbanas tienen acceso limitado a alimentos frescos y sostenibles.

---

# Solución 

BloomChain propone un sistema hidropónico horizontal inteligente que:

* Reduce hasta un 90% el consumo de agua.
* Aprovecha espacios pequeños.
* Automatiza el monitoreo mediante sensores IoT.
* Registra datos seguros utilizando blockchain.
* Permite trazabilidad completa de cada planta.

Nuestro sistema facilita la agricultura urbana sostenible utilizando tecnología moderna, accesible y eficiente.
